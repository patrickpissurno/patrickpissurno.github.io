Welcome to JSGameIDE

JSGameIDE is both an IDE and a library, made to help you to create fabulous games in Javascript. Our objective is to provide you a reliable tool with a fast workflow and the possibility to create a high quality final product, for free.

Want to help?

If you did like the project and want to improve the general user experience with it, feel free to fork and develop. If the result turns out a good product, I'll add it to the main version (as an update) and put your name into the credits.
