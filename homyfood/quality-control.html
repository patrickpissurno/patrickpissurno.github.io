Controle de qualidade
Só os melhores passam no nosso rígido controle

Seu conforto em primeiro lugar

A gente não tolera falhas. Nosso processo de controle de qualidade é extremamente rígido para garantir o melhor conforto para você. Todos os nossos vendedores cuidam do preparo, pois sabem que é te agradando que você volta a comprar com eles.

Como é feito

Todos os detalhes do processo são feitos de maneira automatizada, impedindo qualquer erro humano. Funciona da seguinte maneira:

  1. Cadastro
    Nosso sistema garante que cada vendedor pode criar apenas uma conta. Isso garante que sempre responderá por todos os seus feitos (bons ou ruins).
  2. Feedback real
    Na tela de todos os pratos do app é possível ver as 3 tags mais selecionadas pelos outros compradores para descrever a experiência com ele. Desde "delicioso", "quentinho", "bem geladinho" até "salgado", "sem sal" ou até mesmo "cabelo", "formiga".
  3. Só os melhores pratos
    Além das tags, os compradores podem selecionar o nível de satisfação com o que pediram, numa escala de 1 à 5.

    Caso a nota média de um prato específico fique menor que 3, este é automaticamente desabilitado e entramos em contato com o vendedor para entender o que está acontecendo.
  4. E os melhores vendedores
    Caso a nota média de todos os pratos de um determinado vendedor seja menor que 3, este é automaticamente bloqueado de acessar o app. Cabe à ele entrar em contato com a gente para se explicar.
