User Support

Here I'll try to help you to solve your problems with my software.

Before contacting me by e-mail, try these steps:

  1. Check if your smartphone/computer meets all the System Requirements.
  2. For apps/games that need internet, check if you are connected.
  3. Try to reinstall the app/game.
  4. If the game is on beta testing, check if you meet all the requirements to participate of it.

If nothing above did solve your problem, feel free to get in touch with me @ contato@patrickpissurno.com.br. I'll try to solve your problem as fast as possible.

Suporte de Usuário

Aqui irei lhe ajudar a resolver seus problemas com meus softwares.

Antes de me mandar um e-mail, tente o seguinte:

  1. Verifique se seu smartphone/computador cumpre todos os Requesitos de Sistema.
  2. Para Apps/Jogos que precisam de internet, verifique se você está conectado.
  3. Tente reinstalar o produto.
  4. Se o jogo/app está em fase de testes beta, verifique se você cumpre todos os requesitos para participar dele.

Caso nada acima tenha resolvido seu problema, entre em contato: contato@patrickpissurno.com.br. Vou tentar resolver seu problema o mais rápido possível.
